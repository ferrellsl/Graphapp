Colours

The Colour object:

  • A Colour is a structure:

    	struct Colour {
		byte alpha;
		byte red;
		byte green;
		byte blue;
	};
  • The maximum value of any byte is 0xFF (decimal 255) and minimum value is 0.
  • The first byte stores the alpha channel transparency information. 0 means opaque, FF means fully transparent.
  • The next three bytes give the intensity of red, green and blue light.
  • So {0,0xFF,0,0} is bright red, and {0,0x7F,0,0x7F} is dark purple.
  • Colours can be constructed with the rgb function.
